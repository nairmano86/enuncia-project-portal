# Enuncia Project Portal Backend

## Usage

Install dependencies:

```
npm install
```

Start the server:

```
node server.js
```

## API Endpoints

- `GET /api/projects` – Get all projects  
- `POST /api/projects` – Add a project  
- `GET /api/freelancers` – Get all freelancers  
- `POST /api/freelancers` – Register freelancer
